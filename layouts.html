The Box Model

The CSS box model describes how every HTML element is shaped and spaced.

    It has four parts:
  • Content: What's inside the element (like text or images).
  • Padding: The space between the content and the border.
  • Border: The line (visible or not) that wraps around the element.
  • Margin: The space outside the border that separates this element from the others.
