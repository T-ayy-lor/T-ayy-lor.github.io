HTML elements

We've discussed how websites are comprised of various HTML elements and defined by HTML tags.

Typically, HTML elements have an opening tag, and a closing tag.

However, there exist some elements in which there are no closing tags. These are known as void elements.

W3Schools provides a comprehensive list of HTML tags.

HTML structure

When beginning a new website, you must first lay down several "boilerplate" HTML elements that are necessary for allowing your browser to effectively display the content you define.

See helloworld.html for the simplest start template with descriptions that I'd recommend for new web projects.

CSS

Every HTML element has predefined properties that can be manipulated via CSS, allowing for us to control and customize how our HTML elements are displayed.

These properties can be configured via an external .css file, an internal style tag, and an inline style attribute on opening tags.

W3schools again provides a comprehensive (and extensive) list of all CSS properties for the various HTML elements.
