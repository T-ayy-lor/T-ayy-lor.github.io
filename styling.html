Units

For sizing elements, CSS supports both absolute and relative units for sizing elements.

Absolute units, like px, define fixed measurements that do not scale with the page.

Relative units, like %, em, and rem, adjust based on the parent or root font size. Additionally, viewport units, vw and vh, scale with the browser window.

Scaling

All your HTML elements should be styled so that they can be viewed on any type of screen.

A common trick for scaling elements is to set the element's max-width to 100% (any % will do) and it's height to auto, allowing to to scale with it's container while maintaining it's proportions.


See W3schools.

Media Queries

Media Queries apply CSS properties when certain conditions are met.

Their syntax follows @media (property: value), meaning "if that property equals that value, then the contained css is applied".


See W3schools.
