TDA Web Workshop

Firefox

Firefox's inspector tools are an absolute must-have for smooth designing and developing.

VS code's Live Server Extension

Not strictly necessary, but, provides the ability to auto refresh, allowing you to see changes as you make them.

Mozilla Developer Network (MDN)

MDN is considered the industry standard reference for HTML, CSS, and JavaScript.

W3schools

W3schools is an excellent resource for referencing all aspects of web development. For almost every HTML element and every CSS property, W3schools has interactive examples, and thorough descriptions.

CSS-tricks

CSS-tricks is a site filled with various articles, guides, and examples for modern front-end design. It's a good place to scroll through from time-to-time, you might eye a cool styling technique that you'd like to implement.

Josh Comeau

Josh Comeau is a somewhat-famous but very esteemed front end developer, specializing in html/css/js, and provides several free bite-sized tutorials on his website!

He even has a free book on developing portfolio websites!

Front End Mentor

If you're struggling with inspiration, but still want to excercise your front end skills, front end mentor has many front end design challenges.

Awwwards

Awwwards is the site that gives awards to the best web designs. Great place to find inspiration!

Colorhunt

Great space for finding cohesive color palettes.

Github Pages

For deploying your website, I'd recommend Github Pages. It's free, and it gives you some github xp. Note, it only supports "static html", like what we've covered today. No backend.
